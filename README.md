## Lepaya Courses

[Nest](https://github.com/nestjs/nest) with [TypeScript](https://www.typescriptlang.org/) are used for this project.

## Installation
Install the dependencies using [NPM](https://www.npmjs.com/)

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
